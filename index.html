How to Run BLAST

This tool creates command-line BLAST+ commands.

First verify BLAST is installed: 

blastn -version

Create a BLAST Database

Linux / WSL: 

makeblastdb \
-in genome.fasta \
-dbtype nucl \
-out genome_DB

Windows Command Prompt: 

makeblastdb ^
-in genome.fasta ^
-dbtype nucl ^
-out genome_DB

PowerShell: 

makeblastdb `
-in genome.fasta `
-dbtype nucl `
-out genome_DB

BLAST Parameters

E-value: Determines statistical significance. Lower values are stricter.

Word Size: Controls initial matching seed size.

Percent Identity: Controls sequence similarity requirement.

CPU Threads: Controls processor usage.

Output Fields

qseqid - Query sequence ID
sseqid - Subject/database sequence ID
pident - Percent identity
length - Alignment length
mismatch - Number of mismatches
gapopen - Number of gaps
qstart/qend - Query coordinates
sstart/send - Subject coordinates
evalue - Statistical significance
bitscore - Alignment score
qseq/sseq - Aligned sequences

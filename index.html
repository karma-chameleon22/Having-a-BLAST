About BLAST

BLAST compares biological sequences against a database to identify similar sequences.

Parameters

E-value: Controls how statistically significant a match must be. Lower values are more stringent.

Word Size: Controls the initial matching seed length. Smaller values increase sensitivity and are useful for short sequences.

Percent Identity: Controls how similar sequences must be. 100 means only identical matches are accepted.

CPU Threads: Controls how many processor cores BLAST uses.

Output Fields

Common BLAST output options:

  • qseqid - query ID
  • sseqid - subject ID
  • pident - percent identity
  • length - alignment length
  • evalue - significance
  • bitscore - alignment score
  • qseq/sseq - aligned sequences

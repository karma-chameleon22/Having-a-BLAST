How to Run BLAST

This tool generates commands for NCBI BLAST+. Install BLAST+ before running generated commands.

Check Installation

blastn -version

Create a Database

makeblastdb \
-in genome.fasta \
-dbtype nucl \
-out genome_DB

Copy the generated command into your terminal.

BLAST Parameters

Controls how likely a match occurred by chance.

Lower values = stricter searches.

Examples:
1e-3 → relaxed
1e-7 → common high confidence search
1e-20 → very strict
Controls the initial matching sequence length.

Smaller values:
• More sensitive
• Better for short sequences

Common short sequence values: 7-11
Controls how similar sequences must be.

100 = exact matches only
95 = allows small differences
Controls how many processor cores BLAST uses.

More threads usually speed up searches.

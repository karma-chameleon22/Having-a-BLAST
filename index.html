How to Run BLAST

This tool creates command-line BLAST commands for NCBI BLAST+.

Before running generated commands, confirm BLAST is installed: 

blastn -version

Create a BLAST Database

If you do not already have a database, create one using: 

makeblastdb \
-in genome.fasta \
-dbtype nucl \
-out genome_DB

BLAST Parameters

E-value: Controls the likelihood that a match occurred by chance. Lower values are more stringent.

Word Size: Controls the initial matching seed length. Smaller values increase sensitivity.

Percent Identity: Controls how similar sequences must be. 100 means only exact matches.

CPU Threads: Controls how many processor cores BLAST uses.

Output Fields

qseqid - Query sequence ID
sseqid - Database sequence ID
pident - Percent identity
length - Alignment length
evalue - Statistical significance
bitscore - Alignment score
qseq/sseq - Aligned sequences

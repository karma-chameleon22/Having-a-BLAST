What is BLAST?

BLAST (Basic Local Alignment Search Tool) compares biological sequences against a database to identify similarities between DNA, RNA, or protein sequences.

Common BLAST Programs

  • blastn - nucleotide sequence against nucleotide database
  • blastn-short - short nucleotide sequences such as tRFs or miRNAs
  • blastp - protein sequence comparisons
  • blastx - translated nucleotide query against protein database
  • tblastn - protein query against translated nucleotide database

Before Running BLAST

Install NCBI BLAST+ and verify installation: 

blastn -version

Create a BLAST Database

makeblastdb \
-in sequences.fasta \
-dbtype nucl \
-out database_name

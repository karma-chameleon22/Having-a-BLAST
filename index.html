How to Run BLAST

BLAST compares biological sequences against a database. Install NCBI BLAST+ before using generated commands.

Check BLAST Installation

blastn -version

Create a Database

makeblastdb \
-in genome.fasta \
-dbtype nucl \
-out genome_DB

Copy the generated command into your terminal.
